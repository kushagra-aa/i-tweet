# I Tweet

<img src="./assets/logo.png" width="15%" alt="logo"/>

I Tweet is a Twitter Clone.
It has Twitter Login, comments, retweet and dark mode.
It is made with NextJS, TypeScript, Sanity and TailwindCSS.
